# file-encryption-tool
I made this tool using Python and PyQt5 for GUI. You can try it out!

# Commands
To run this file use ``python file_encryption_tool.py``

## How to Use

1. **Launch:** Run `python main.py`.
2. **Setup:** If you don't have keys, click **Generate New Keys** and select a folder.
3. **Load:** Click **Load Public Key** or **Load Private Key** to load existing keys.
4. **Process:**
    - Click **Select File to Process**.
    - Click **Encrypt File** to create a `.vault` version.
    - Click **Decrypt File** to restore a `.vault` file to its original state.
